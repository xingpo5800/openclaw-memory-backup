# 规则索引

- **用途**：适用于任何类型任务的通用执行准则
- **重要性**：最高级别，是所有行为的基础
- **更新频率**：根据使用情况持续优化

### 📋 系统规则
- **文件**：`system-rules.md`
- **用途**：系统运行和行为的基本规则
- **重要性**：高，确保系统稳定运行
- **更新频率**：根据系统需求调整

## 规则使用优先级

1. **通用高效执行准则** - 任何任务都要首先遵循
2. **系统规则** - 系统运行必须遵守
3. **通信规则** - 交互时必须遵守
4. **Token节省规则** - 在处理大量信息时使用
5. **动画脚本规则** - 在创作动画内容时使用

## 规则应用流程

### 任务开始前
1. 查阅通用高效执行准则
2. 根据任务类型查阅相关规则
3. 制定具体执行计划

### 任务执行中
1. 按照准则执行
2. 遇到问题查阅相应规则
3. 根据准则调整执行方式

### 任务完成后
1. 总结经验教训
2. 更新相关规则
3. 记录改进方向
