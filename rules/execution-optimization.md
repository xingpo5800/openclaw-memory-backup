# 执行优化规则

## 触发条件

- 遇到执行效率问题时自动触发
- 发现规则冲突时自动触发
- 遇到复杂问题需要优化方案时触发

## 优化目标
- 减少不必要的工具调用
- 增强执行效率
- 解决规则冲突
- 促进进化

## 智能记忆检索
- 任务开始前 → 先搜记忆有没有类似经验
- 任务进行中 → 遇到问题也搜记忆
- 长的经验 → 建专项文件，按需调用

## 精简执行
- 合并连续read操作
- 用memory_get替代部分read
- 简单问题直接回答，不需要加载一堆规则
- 复杂问题按需加载规则

## 规则冲突解决
1. 发现冲突
2. 评估影响
3. 选优先级或更新
4. 更新前确认逻辑

## 一句话记住
**规则是基石，进化是目标，两者相辅相成。**
