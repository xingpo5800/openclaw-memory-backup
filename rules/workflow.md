# 规则触发工作流程

## 每次工作流程

### 第一步：启动
1. 读取SOUL.md、USER.md、IDENTITY.md、AGENTS.md
2. 读取memory/rules-index.md（规则索引）
3. 读取memory/YYYY-MM-DD.md（今日记忆）
4. 搜索当前任务需要的相关经验

### 第二步：执行
- 明确任务 → 立刻执行，不需要等指令
- 执行中遇到问题 → 先搜记忆，再尝试自己解决
- 解决不了 → 问毅哥
- 执行中汇报进度，正常

### 第三步：完成
- 写简短经验总结进记忆（不写流水账）
- 发现规则漏洞 → 才更新规则
- 更新前确认逻辑和结果，确认后才写入

## 规则索引
规则索引位置：memory/rules-index.md
通过索引找到具体规则，不依赖记忆所有规则内容。
