# OpenClaw 工作区组织结构

## 目录结构规划

### 核心配置目录
- **SOUL.md** - AI助手个性设定
- **USER.md** - 用户信息记录  
- **IDENTITY.md** - 身份信息定义
- **TOOLS.md** - 本地工具配置
- **AGENTS.md** - Agent核心功能手册
- **MEMORY.md** - 记忆系统索引

### 核心功能目录

#### 1. 记忆系统 (memory/)
- **YYYY-MM-DD.md** - 每日记忆记录
- **rules-index.md** - 规则索引文件
- **rules/** - 规则文件存储
  - system-rules.md - 系统规则
  - token-saver.md - 省tokens规则
  - files-manager.md - 文件操作规则
  - animation-script.md - 动画脚本规则
  - communication.md - 通信规则

#### 2. 项目管理 (projects/)
- 项目开发目录
- 代码仓库管理
- 任务跟踪

#### 3. 知识管理 (knowledge/)
- 学习资料
- 技术文档
- 知识库

#### 4. 工具配置 (tools/)
- 外部工具配置
- 插件设置
- 系统工具

#### 5. 脚本执行 (scripts/)
- 自动化脚本
- 工具脚本
- 执行文件

#### 6. 会话管理 (sessions/)
- 持久化会话
- 会话历史
- 跨会话通信

### 记忆加载逻辑

#### 启动流程
1. **读取配置文件**：SOUL.md → USER.md → IDENTITY.md
2. **检查记忆系统**：AGENTS.md → 记忆系统初始化
3. **读取今日记忆**：memory/YYYY-MM-DD.md
4. **读取规则索引**：memory/rules-index.md

#### 执行流程
1. **指令分析**：memory_search匹配规则
2. **规则执行**：读取并执行匹配规则
3. **操作执行**：执行具体任务
4. **记忆更新**：记录新信息到记忆系统

### 规则文件组织

#### 规则类型
- **系统规则**：启动流程、执行流程、错误处理
- **操作规则**：文件读写、会话管理、外部操作
- **通信规则**：消息格式、回复策略、状态通知
- **技能规则**：技能调用、组合规则、错误处理

#### 规则索引 (rules-index.md)
```markdown
# 规则索引

## 操作规则
- system-rules.md - 系统核心规则
- token-saver.md - 省tokens规则
- communication.md - 通信规则
- animation-script.md - 动画脚本规则
```

### 工作区约定

#### 文件命名规范
- 使用小写字母和下划线命名
- 日期使用YYYY-MM-DD格式
- 规则文件使用.md后缀

#### 访问权限
- **可公开访问**：配置文件、文档、脚本
- **受限访问**：记忆文件、会话数据、敏感信息
- **仅管理员**：系统配置、工具设置

#### 数据流向
```
用户请求
    ↓
记忆搜索 (memory_search)
    ↓
规则匹配 (优先级：高→中→低)
    ↓
执行操作 (对应规则)
    ↓
结果反馈
    ↓
记忆更新 (新信息存入)
```

### 扩展机制

#### 新功能添加流程
1. 创建功能规则文件（在rules/目录）
2. 更新rules-index.md索引
3. 在相关目录创建对应文件夹
4. 更新工作区文档

#### 第三方集成
- 通过tools/目录管理外部工具
- 通过scripts/目录管理可执行脚本
- 通过knowledge/目录管理外部文档
